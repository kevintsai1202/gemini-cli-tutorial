# GEMINI.md

## 目錄概觀

此目錄包含名為「使用Gemini CLI Vibe Coding的魅力：從 AI 協作到全自動開發」課程的相關教材。本課程旨在教導如何運用 Gemini CLI 進行高效的開發工作，並整合 `ffmpeg`、`pandoc` 與 Chrome Devtools 等外部工具。

## 關鍵檔案

*   `課綱.md`: 課程大綱，詳細說明課程的四個部分、學習目標和時間分配。
*   `教學文件.md`: 課程的詳細教材，為每個章節提供解說、安裝步驟和實例。
*   `AGENTS.md`: 定義了 AI 代理人（我）在此專案中的工作環境、文件標準、程式碼規範和任務管理流程。重點包括：
    *   **環境**: 使用繁體中文、PowerShell 7+ 相容指令，並指定 Java 路徑。
    *   **文件**: 在進行任何修改前，需先更新規格文件 (`spec.md`, `api.md`)。
    *   **任務**: 開發前需在 `todolist.md` 中進行任務拆分，並追蹤任務進度。
*   `SOP/`: 此目錄存放安裝所需軟體的標準作業程序（SOP），例如 `nodejs_installation_sop.md` 和 `python_installation_sop.md`。
*   `gemini-cli-tips-and-tricks/`: 完整的使用技巧指南（分為多個部分），提供約 30 個專業技巧，以有效利用 Gemini CLI 進行代理式編碼（Agentic Coding）。

## 使用方式

此目錄是 Gemini CLI 課程所有教材的中央儲存庫，可作為講師和學員的參考資料。`AGENTS.md` 檔案為我提供了與此專案互動的具體指示，強調結構化和文件化的工作流程。
