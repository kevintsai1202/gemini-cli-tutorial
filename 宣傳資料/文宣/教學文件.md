# 課程名稱：用 AI 幫你寫程式！學會 Gemini CLI 打造 4 大超實用自動化流程

本課程將帶您深入 Gemini CLI 的世界，從基礎指令到進階應用，學習如何將 AI 融入您的開發工作流程，最終實現高效的「Vibe Coding」開發模式。

**課程時長：** 2 小時 (120 分鐘)

**課程亮點：**

*   **應用程式化思維：** 不只學習單一指令，更重要的是學習如何組合不同的工具，將您的想法打造成一個個實用的應用程式。
*   **原型驅動開發：** 學習使用 Gemini CLI 作為您的核心開發引擎，快速驗證想法，並將其部署為可執行的應用程式。
*   **跨工具整合：** 學習如何讓 Gemini 成為您開發工作流程的指揮中心，協同 `ffmpeg` (影音處理)、`pandoc` (文件轉換)、Chrome Devtools MCP(網頁分析) 等專業工具，高效完成複雜專案。

**事前準備：**
*   **VS Code:** 請確保您的開發環境已安裝 [VS Code](https://code.visualstudio.com/Download) 目前最新1.105版。
*   **Node.js:** 請確保您的開發環境已安裝 [Node.js](https://nodejs.org/) 22.x 或更高版本。
*   **Python:** 請確保您的開發環境已安裝 [Python](https://www.python.org/) 3.x 或更高版本。

## 第一部分：基礎奠基與核心概念 (20 分鐘)

### 歡迎與介紹 (5 分鐘)

**為什麼要學 Gemini CLI？**

在傳統的開發模式中，我們需要手動編寫每一行程式碼。然而，隨著 AI 技術的發展，我們可以將許多重複性、規律性的工作交給 AI 處理。Gemini CLI 不僅僅是一個 AI 助理，它更像是一位全自動的開發夥伴，能夠理解您的意圖，並自動執行複雜的任務，從而極大地提升您的開發效率。

**成果展示：**

在本課程中，我們將會完成以下幾個小專案：

1.  **自動更新腳本：** 使用 Gemini CLI 撰寫一個能夠自動更新 CLI 環境的批次腳本。
2.  **頻道分析報告：** 自動蒐集指定 YouTube 頻道的資訊，找出相似的頻道進行比較，並最終產出詳細的比較報告。
3.  **YT 影片字幕產生器：** 提供一個 YouTube 影片連結，Gemini CLI 將自動轉換為字幕檔並生成影片摘要。

### 環境建置：你的第一個 Gemini 指令 (15 分鐘)

**「萬能工具箱」概念：**

我們可以將 Gemini CLI 想像成我們開發工作流程的「大腦」，它負責思考、規劃和指揮。然而，要完成實際的任務，我們還需要「手」和「腳」，也就是各種專業的工具。本次課程中，我們將重點介紹以下幾款如同瑞士刀般強大的工具：

*   **Gemini CLI:** 我們的大腦，負責與 AI 溝通。
*   **pandoc:** 一款強大的文件轉換工具，可以將檔案在不同格式之間進行轉換。
*   **ffmpeg:** 一款開源的影音處理工具，可以進行轉檔、剪輯、擷取音訊等操作。

**安裝步驟：**

1.  **Gemini CLI 安裝：**
    開啟您的終端機，並執行以下指令來安裝 Gemini CLI：
    ```bash
    npm install -g @google/gemini-cli
    ```

2.  **pandoc 安裝與介紹：**
    *   **介紹：** Pandoc 是一個通用的文件格式轉換工具，支援數十種文件格式，例如 Markdown, HTML, DOCX, PDF 等。
    *   **安裝：** 請參考 [pandoc 官方網站](https://pandoc.org/installing.html) 的說明進行安裝。

3.  **ffmpeg 安裝與介紹：**
    *   **介紹：** FFmpeg 是一套完整的跨平台影音處理函式庫，能夠錄製、轉換、串流音訊和影片。
    *   **安裝：** 請參考 [ffmpeg 官方網站](https://ffmpeg.org/download.html) 的說明進行安裝。

### 核心語法：與 Gemini 高效對話 (5 分鐘)

*   **基本 Prompt:** 直接在終端機輸入您想對 Gemini 說的話，例如 `gemini "你好嗎？"`。
*   **管道 (Pipe):** 您可以使用 `|` 或 `>`(windows 為'<') 將其他指令的輸出結果傳遞給 Gemini CLI，實現更複雜的工作流程。

## 第二部分：活用 MCP 工具：給 Gemini 擴充功能 (40 分鐘)

### 安裝 MCP 工具

*   **介紹：** MCP (Multi-modal Co-pilot) 是 Gemini CLI 的一個強大擴充功能，它允許 Gemini 控制您的 Chrome 瀏覽器，執行網頁操作、擷取資訊等任務。您可以從 [geminicli.com](https://geminicli.com) 了解更多資訊。
*   **安裝：**
    1.  在 Chrome 瀏覽器中，安裝 [Chrome DevTools Extension](https://chrome.google.com/webstore/category/extensions)。 (此處應提供實際擴充功能的連結)
    2.  透過 Gemini CLI 連接並授權。

### 如何透過提示詞控制 MCP

您可以透過自然的語言提示詞來控制 MCP。例如，您可以要求 Gemini：

*   「打開 YouTube 網站」
*   「在搜尋框中輸入『Gemini CLI』並按下 Enter」
*   「擷取目前頁面的所有標題和連結」

**實作：頻道分析報告**

我們將利用 MCP 來自動蒐集 YouTube 頻道的資訊，並與其他類似頻道進行比較，最終產出網頁及報告。

### 將 pandoc 指令包裝成斜線指令

為了更方便地使用 pandoc，我們可以將其常用的指令包裝成 Gemini CLI 的斜線指令 (Slash Command)。例如，我們可以建立一個 `/convert` 指令，讓我們可以透過對話的方式，輕鬆地將網頁轉換為 doc 文件。

## ⭐ 第三部分：結合 ffmpeg、whisper 開發一套字幕產生器：Vibe Coding 實作 (45 分鐘)

本章節是課程的精華，我們將透過 Gemini CLI，以「Vibe Coding」的模式開發一個 YouTube 影片字幕產生器。

### 準備工作

1.  **Groq API Key:** 我們將使用 [Groq](https://groq.com/) 提供的 Whisper 模型來進行語音轉文字。Groq 提供了免費的 API 額度，請先申請您的 API Key。
2.  **設定環境變數：** 將您的 Groq API Key 設定為環境變數，以便 Gemini CLI 可以取用。

### 開發流程

1.  **擷取音訊：** 使用 `ffmpeg` 從 YouTube 影片中擷取音訊，並轉換為較小的音檔格式 (例如 mp3)，以避免超過 API 的檔案大小限制。
2.  **語音轉文字：** 呼叫 Groq 的 Whisper API，將音訊檔案轉換為逐字稿。
3.  **生成字幕檔：** 將逐字稿處理成 `.srt` 或 `.vtt` 格式的字幕檔。
4.  **創意發揮：**
    *   **影片摘要：** 讓 Gemini 閱讀逐字稿，並自動生成影片的重點摘要。
    *   **社群貼文：** 將影片內容轉換為適合在不同社群平台 (如 Facebook, Twitter, Instagram) 發布的貼文。
