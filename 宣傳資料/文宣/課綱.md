# 課程名稱：使用Gemini CLI Vibe Coding的魅力：從 AI 協作到全自動開發
課程時長： 2 小時 (120 分鐘)
課程亮點：
應用程式化思維： 不只學指令，更可學習如何搭配各種工具，親手打造成實用 App。
原型驅動開發： 學習使用 CLI 作為核心引擎，快速驗證想法，並將其部署為應用程式。
跨工具整合： 學習如何讓 Gemini 成為指揮中心，搭配 ffmpeg (影音)、pandoc (文件)、Chrome Devtools MCP等專業工具，協同完成專案。
事前準備：環境需先安裝好 VS Code、NodeJS 20+(目前最新版本22.x)、Python 3+(待確認)

## 第一部分：基礎奠基與核心概念 (20 分鐘)
歡迎與介紹 (5 分鐘)
為什麼要學 Gemini CLI？（從 AI 助理，提升到全自動開發助理）。
成果展示： Demo 最終完成品：
1. Gemini CLI 撰寫的自動更新腳本
2. 自動蒐集頻道資訊, 並找出類似頻道進行比較, 最終產出比較報告
3. 提供YT連結, 自動轉成字幕檔以及影片摘要

環境建置：你的第一個 Gemini 指令 (15 分鐘)
安裝：⭐ 介紹「萬能工具箱」概念： Gemini 是大腦，我們需要手腳。重點介紹本次課程的瑞士刀：pandoc, ffmpeg。
1. Gemini CLI 安裝
2. pandoc 安裝及介紹
3. ffmpeg 安裝及介紹

核心語法：與 Gemini 高效對話 (5 分鐘)
基本 Prompt、管道 (| 和 <)、關鍵參數。

## 第二部分：活用 MCP 工具：給 Gemini 擴充功能(40 分鐘)
安裝 MCP 工具：介紹 MCP 以及最新的 Extension 擴展插件
1. 展示 geminicli.com
2. 安裝 Chrome devtools Extension

如何透過提示詞控制 MCP
完成使用MCP工具蒐集頻道資訊並與其他類似頻道進行比較，產出網頁及報告

將 pandoc 指令包裝成斜線指令，透過對話將網頁轉成 doc 文件

## ⭐ 第三部分：結合 ffmpeg、whisper　開發一套字幕產生器：Vibe Coding 實作 (45 分鐘)
本章節是課程的精華，透過 Gemini CLI 開發一個YT影片字幕產生器(可擴充摘要及生成其他平台貼文)。
準備工作
1. groq API key: groq 提供 whisper 模型以及免費額度
2. 透過 ffmpeg 擷取影片並轉換為音檔(檔案較小,避免超過額度)
3. 透過 whisper 轉換逐字稿及字幕
4. 發揮自己的創意將逐字稿轉為摘要、其他平台貼文...
