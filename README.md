# 用 AI 幫你寫程式！Gemini CLI 完全教學

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://github.com)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

一個精美的互動式教學網站，幫助您在 2 小時內掌握 Gemini CLI 的核心技能，打造 4 大超實用自動化流程。

## 📚 課程內容

### 第一部分：基礎奠基與核心概念 (20 分鐘)
- 歡迎與介紹
- 環境建置：你的第一個 Gemini 指令
- 無頭模式：與 Gemini 高效對話
- Gemini 對話參數與熱鍵

### 第二部分：活用 MCP 工具 (40 分鐘)
- 安裝 MCP 工具
- 工具整合應用
- 使用 MCP 蒐集頻道資訊
- 找出相似頻道並產出報告

### 第三部分：結合 ffmpeg、whisper 開發字幕產生器 (45 分鐘)
- 準備工作（取得 Groq API Key）
- 開發流程
- 字幕產生器功能需求實作

### 第四部分：Vibe Coding 的迷失與未來展望 (15 分鐘)
- 不懂程式也能 Vibe Coding?
- AI 未來發展
- Q&A

## ✨ 功能特色

### 🎨 精美的視覺設計
- 深色主題配色，專業且護眼
- 漸層背景與卡片式設計
- 流暢的動畫效果與過渡

### 📱 互動式側邊欄導航
- 固定側邊欄，方便快速跳轉
- 即時顯示學習進度（百分比）
- 自動高亮當前閱讀的章節
- 顯示每個章節的預計時長

### ⏱️ 學習計時器
- 精確的學習時間追蹤
- 今日學習時間統計
- 總學習時間累計
- 資料持久化儲存

### 💻 程式碼高亮與複製
- 使用 Prism.js 進行語法高亮
- 一鍵複製程式碼功能
- 支援 Bash、Python、JavaScript 等語言
- 複製成功有視覺回饋

### 📊 學習進度追蹤
- 每個章節都有「標記為已完成」按鈕
- 進度自動儲存到 localStorage
- 刷新頁面後進度不會遺失
- 視覺化進度條顯示

### 📱 響應式設計
- 支援桌面與行動裝置
- 自動調整佈局
