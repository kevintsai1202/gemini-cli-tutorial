## 技巧 27：關注路線圖（背景代理等）

**快速用例：** 隨時了解即將推出的 Gemini CLI 功能——透過關注公開的 Gemini CLI 路線圖，您將在主要計畫增強功能（例如用於長時間任務的背景代理）到來之前了解它們，從而讓您能夠規劃並提供回饋。

Gemini CLI 正在迅速發展，新版本頻繁發布，因此追蹤即將推出的功能是明智之舉。Google 在 GitHub 上維護著 Gemini CLI 的公開路線圖，詳細說明了近期未來的關鍵重點領域和目標功能。這本質上是一個活文件（和一系列問題），您可以在其中查看開發人員正在處理什麼以及正在進行什麼。

例如，路線圖上一個令人興奮的項目是支援背景代理——生成自主代理的能力，這些代理在背景中運行以持續或非同步地處理任務。根據路線圖討論，這些背景代理將允許您將長時間運行的進程委派給 Gemini CLI，而無需綁定您的互動式會話。您可以，例如，啟動一個背景代理，該代理監控您的專案以查找某些事件或定期執行任務，無論是在您的本機機器上還是透過部署到 Cloud Run 等服務。此功能旨在「從 CLI 啟用長時間運行、自主任務和主動協助」，本質上擴展了 Gemini CLI 的實用性，而不僅僅是按需查詢。
